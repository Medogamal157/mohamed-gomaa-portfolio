# Mohamed Gomaa Portfolio

Static-first portfolio for Mohamed Gomaa.

## What this version prioritizes

- Result-focused case studies
- Clear contribution wording for team/shared projects
- Mobile-first responsive design
- SEO metadata and JSON-LD Person schema
- Accessibility basics: skip link, semantic sections, focus states, reduced motion
- Lightweight optional canvas animation — no heavy 3D dependency

## Local preview

```bash
python3 -m http.server 4173
```

Open: `http://127.0.0.1:4173`

## Deploy to Vercel

This is a static site. The simplest deployment is to push this folder to GitHub and import it into Vercel as a static project.

After deployment, update the canonical and Open Graph URL in `index.html` from the placeholder to the final Vercel URL.
